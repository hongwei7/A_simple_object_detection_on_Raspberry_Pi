# Object_Detection_on_pi
A simple object detection on RaspberryPi(using Inspection V3 based on TensorFlow)</br>
# CHN
## 安装依赖</br>
  tensorflow</br>
## 设置路径</br>
在run.py第31行设置路径</br>
## 运行
在bash中运行run.py</br>
<code>python3 run.py</code></br>
标签已经过有道翻译机翻，支持一千多种物体检测，数据基于ImageNet。
